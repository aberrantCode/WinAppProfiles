# Application Icons

This folder contains bundled application icons that can be selected as alternatives to extracted icons.

## Supported Formats
- PNG (preferred, with transparency)
- ICO
- BMP

## Naming Convention
Icons should be named descriptively, e.g.:
- `visual-studio.png`
- `docker.png`
- `sql-server.png`
- `chrome.png`

## Icon Size
Recommended size: 64x64 pixels or larger (will be scaled automatically)
